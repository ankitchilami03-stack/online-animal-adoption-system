# Online Animal Adoption & Donation System

## Project Description

This project is a web-based platform that allows users to adopt animals online and donate money for their care. The system helps support cowsheds and zoos by collecting donations that are used to feed and take care of animals.

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL

## Features

* User registration and login
* Online animal adoption system
* Donation feature to support cowsheds and zoos
* Online payment integration for donations
* Animal care support such as feeding and maintenance
* Admin management for animals and donations

## Purpose

The main goal of this project is to help protect and care for animals by allowing people to contribute financially and support their food and shelter.

## How to Run the Project

1. Install XAMPP or any PHP server.
2. Import the database file into phpMyAdmin.
3. Place the project folder in the `htdocs` directory.
4. Run the project in the browser using localhost.
